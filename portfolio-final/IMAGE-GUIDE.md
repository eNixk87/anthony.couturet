## HOW TO EXPORT IMAGES

### From PDFs:
1. Open PDF in Preview (Mac) or Acrobat
2. Zoom to 100% or higher
3. Use screenshot tool (Cmd+Shift+4 on Mac, Snipping Tool on Windows)
4. Save as JPG at 80% quality
5. Aim for **1600px wide minimum**

### From the Chemin website zip:
1. Extract `web_design.zip`
2. Open each `.html` file in Chrome
3. Screenshot at full width (1440px+)
4. For mobile: use Chrome DevTools (F12) → toggle device toolbar → select iPhone/Pixel
5. Screenshot the mobile view

### Tips:
- Hero/cover images work best at **16:9 or wider**
- Detail crops can be **1:1 or 4:3**
- All images should be JPG (not PNG) for file size
- Run through [TinyJPG](https://tinyjpg.com/) to compress before uploading
